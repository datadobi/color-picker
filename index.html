Color Picker

Demos for <color-picker>

Basic

<color-picker></color-picker>

Initial value & previous value

Use value for the initial color and previous-value to show a reset target. Hover and click the color circle to reset.

<color-picker previous-value="#ff00bf44" value="hsla(135,100%,50%,0.5)"></color-picker>

Palettes

Set palettes to an array of arrays of CSS color strings.

CSS Custom Properties

value and previous-value accept CSS custom property names.

Disable alpha

disable-alpha + disable-sl

Disable color formats

disable-hex

Small theme

<color-picker theme="small"></color-picker>

Pinned inputs & palettes

pinned-inputs

pinned-palettes

Custom resolution

Use step-alpha and step-hsl for finer precision.
